1. Introduction to C

Overview of C programming, its history, setup, and writing the first C program.

2. Basic Syntax and Structure

Keywords, identifiers, comments, variables, constants, and control flow statements.

3. Variables and Data Types

Primitive and derived data types, constants, type conversion, and size of data types.

4. Operators in C

Arithmetic, relational, logical, bitwise, assignment, and conditional operators.

5. Control Structures

If-else, switch-case, loops (for, while, do-while), and break/continue.

6. Functions in C

Function declaration, definition, recursion, passing parameters, and using library functions.

7. Pointers

Introduction to pointers, pointer arithmetic, arrays and pointers, and dynamic memory allocation.

8. Arrays

Array declaration, initialization, multi-dimensional arrays, and array manipulation.

9. Strings

Working with strings, string functions, and string manipulation in C.

10.Sorting and Searching

Bubble sort, Insertion sort, Selection sort and Binary search, Linear search in C

11. File Handling in C

File operations (read, write, append), file opening modes, and file handling functions.

12. Structure, Union and Enum in C

Learn to manage data efficiently in C with Structures, save memory using Unions, and improve code readability with Enums in this concise course.

13. Important Coding Questions

Essential practice problems on loops, arrays, and functions.

14. Mini Projects with Solutions

Practical projects including solutions to reinforce learning.

15. Assessment of course

Test your knowledge and earn a certificate.
