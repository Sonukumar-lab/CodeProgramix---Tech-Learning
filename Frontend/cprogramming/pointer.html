Pointers in C

A pointer in C is a variable that stores the memory address of another variable. It allows us to directly access and modify the values stored in memory, which makes pointers powerful tools for efficient memory management and manipulation. Understanding pointers is crucial for low-level programming, as they provide direct access to memory and allow you to work with dynamic memory, arrays, structures, and function arguments efficiently.

1. Introduction to Pointers

A pointer is a variable that holds the memory address of another variable. It allows indirect manipulation of variables, enabling functions to modify variables defined in other parts of the program.

Pointer Declaration

To declare a pointer in C, we use the * symbol before the pointer variable's name. The type of the pointer must match the type of the variable it points to.

type *pointer_name;
  • type: The type of variable the pointer points to (e.g., int, char, float).
  • pointer_name: The name of the pointer.

Example of Pointer Declaration

#include <stdio.h>
int main() {
    int a = 10;
    int *ptr;  // pointer to int
    ptr = &a;  // ptr holds the address of a
    printf("Value of a: %d\n", a);
    printf("Address of a: %p\n", (void *)&a);
    printf("Value at ptr: %d\n", *ptr);  // dereferencing the pointer
    return 0;
}

2. Pointer Arithmetic

Pointer arithmetic refers to performing operations like addition or subtraction on pointers. When you add an integer to a pointer, the pointer moves forward by that many memory units, depending on the type it points to.

Example: Pointer Arithmetic

#include <stdio.h>
int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr;
    printf("Value at ptr: %d\n", *ptr);  // Points to arr[0]
    
    ptr++;  // Move the pointer to the next element
    printf("Value at ptr: %d\n", *ptr);  // Points to arr[1]
    
    ptr += 2;  // Move the pointer forward by 2 elements
    printf("Value at ptr: %d\n", *ptr);  // Points to arr[3]
    
    return 0;
}

3. Pointers and Arrays

In C, arrays and pointers are closely related. An array name is essentially a pointer to the first element of the array. You can use pointers to traverse arrays.

Example: Pointers and Arrays

#include <stdio.h>
int main() {
    int arr[] = {10, 20, 30, 40, 50};
    int *ptr = arr;  // Array name is a pointer to the first element
    
    for (int i = 0; i < 5; i++) {
        printf("Value at arr[%d]: %d\n", i, *(ptr + i));  // Pointer arithmetic
    }
    return 0;
}

4. Pointers to Functions

Pointers can also point to functions. This allows you to call a function indirectly through a pointer.

Example: Pointers to Functions

#include <stdio.h>
void greet() {
    printf("Hello, World!\n");
}
int add(int a, int b) {
    return a + b;
}
int main() {
    void (*func_ptr)() = greet;  // Pointer to function
    func_ptr();  // Calling greet() through pointer
    
    int (*add_ptr)(int, int) = add;  // Pointer to add function
    printf("Sum: %d\n", add_ptr(5, 3));  // Calling add() through pointer
    
    return 0;
}

Dry Run:

  • func_ptr is a pointer to the greet function. It is called indirectly using func_ptr().
  • add_ptr is a pointer to the add function. It is called with arguments 5 and 3 using add_ptr(5, 3).

Output:

Hello, World!
Sum: 8

5. Pointers to Structures

You can use pointers to manipulate structures in C. Using pointers to structures allows efficient access and modification of structure members.

Example: Pointers to Structures

#include <stdio.h>
struct Point {
    int x;
    int y;
};
int main() {
    struct Point p1 = {10, 20};
    struct Point *ptr = &p1;  // Pointer to structure
    
    printf("Point p1: (%d, %d)\n", ptr->x, ptr->y);  // Access members using pointer
    
    return 0;
}

Dry Run:

  • ptr stores the address of p1.
  • ptr->x accesses p1.x, and ptr->y accesses p1.y using the pointer.

Output:

Point p1: (10, 20)

6. Dynamic Memory Allocation

Pointers are used for dynamic memory allocation in C, using functions like malloc(), calloc(), and free().

Example: Dynamic Memory Allocation

#include <stdio.h>
#include <stdlib.h>
int main() {
    int *ptr;
    
    // Dynamically allocating memory for an array of 5 integers
    ptr = (int *)malloc(5 * sizeof(int));
    
    if (ptr == NULL) {
        printf("Memory allocation failed\n");
        return 1;
    }
    
    // Assigning values to the allocated memory
    for (int i = 0; i < 5; i++) {
        ptr[i] = i * 10;
    }
    
    // Printing the allocated values
    for (int i = 0; i < 5; i++) {
        printf("Value at ptr[%d]: %d\n", i, ptr[i]);
    }
    
    // Freeing the allocated memory
    free(ptr);
    
    return 0;
}

1.Write a program to swap two numbers using pointers.

#include <stdio.h>
void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

int main() {
    int x, y;
    printf("Enter two numbers: ");
    scanf("%d %d", &x, &y);
    printf("Before swapping: x = %d, y = %d\n", x, y);
    swap(&x, &y);
    printf("After swapping: x = %d, y = %d\n", x, y);
    return 0;
}

2.Write a program to find the largest element in an array using pointers.

#include <stdio.h>
int findLargest(int *arr, int size) {
    int largest = *arr;
    for (int i = 1; i < size; i++) {
        if (*(arr + i) > largest) {
            largest = *(arr + i);
        }
    }
    return largest;
}

int main() {
    int n;
    printf("Enter the number of elements: ");
    scanf("%d", &n);

    int arr[n];
    printf("Enter %d elements: ", n);
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    int largest = findLargest(arr, n);
    printf("The largest element is: %d\n", largest);
    
    return 0;

3.Write a program to reverse an array using pointer arithmetic.

#include <stdio.h>
void reverseArray(int *arr, int size) {
    int *start = arr;
    int *end = arr + size - 1;
    int temp;
    
    while (start < end) {
        temp = *start;
        *start = *end;
        *end = temp;
        start++;
        end--;
    }
}

int main() {
    int n;
    printf("Enter the number of elements: ");
    scanf("%d", &n);

    int arr[n];
    printf("Enter %d elements: ", n);
    for (int i = 0; i < n; i++) {
        scanf("%d", &arr[i]);
    }

    reverseArray(arr, n);

    printf("Reversed array: ");
    for (int i = 0; i < n; i++) {
        printf("%d ", arr[i]);
    }
    printf("\n");

    return 0;
}
