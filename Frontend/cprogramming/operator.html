Operators in C

Operators in C are special symbols used to perform operations on variables and values. They are the building blocks of expressions in C. Understanding operators is essential for performing calculations, comparisons, logical operations, and manipulating data at a bit level.

Types of Operators in C

The operators in C are categorized into several types based on their functionality:

  1. Arithmetic Operators
  2. Relational Operators
  3. Logical Operators
  4. Bitwise Operators
  5. Assignment Operators
  6. Increment and Decrement Operators
  7. Conditional (Ternary) Operator
  8. Sizeof Operator
  9. Comma Operator

1. Arithmetic Operators

Arithmetic operators are used to perform basic mathematical calculations.

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • / : Division
  • %: Modulus (remainder)

Example:

#include <stdio.h>
int main() {
    int a = 10, b = 3;
    int sum = a + b;       // Addition
    int diff = a - b;      // Subtraction
    int product = a * b;   // Multiplication
    int quotient = a / b;  // Division
    int remainder = a % b; // Modulus
    printf("Sum: %d\n", sum);
    printf("Difference: %d\n", diff);
    printf("Product: %d\n", product);
    printf("Quotient: %d\n", quotient);
    printf("Remainder: %d\n", remainder);
    return 0;
}

2. Relational Operators

Relational operators are used to compare two values and return a boolean result: 1 for true and 0 for false.

  • ==: Equal to
  • !=: Not equal to
  • >: Greater than
  • <: Less than
  • >=: Greater than or equal to
  • <=: Less than or equal to

Example:

#include <stdio.h>
int main() {
    int a = 10, b = 5;
    printf("a == b: %d\n", (a == b));  // 0 (false)
    printf("a != b: %d\n", (a != b));  // 1 (true)
    printf("a > b: %d\n", (a > b));    // 1 (true)
    printf("a < b: %d\n", (a < b));    // 0 (false)
    printf("a >= b: %d\n", (a >= b));  // 1 (true)
    printf("a <= b: %d\n", (a <= b));  // 0 (false)
    return 0;
}

3. Logical Operators

Logical operators are used to combine multiple conditions. They work on boolean values and return true (1) or false (0).

  • &&: Logical AND
  • ||: Logical OR
  • !: Logical NOT

Example:

#include <stdio.h>
int main() {
    int a = 1, b = 0;
    printf("a && b: %d\n", (a && b)); // 0 (false)
    printf("a || b: %d\n", (a || b)); // 1 (true)
    printf("!a: %d\n", !a);           // 0 (false)
    return 0;
}

4. Bitwise Operators

Bitwise operators are used to perform bit-level operations on integer types.

  • &: Bitwise AND
  • |: Bitwise OR
  • ^: Bitwise XOR
  • ~: Bitwise NOT
  • <<: Left shift
  • >: Right shift

Example:

#include <stdio.h>
int main() {
    int a = 5, b = 3;
    printf("a & b: %d\n", (a & b));  // Bitwise AND (result = 1)
    printf("a | b: %d\n", (a | b));  // Bitwise OR (result = 7)
    printf("a ^ b: %d\n", (a ^ b));  // Bitwise XOR (result = 6)
    printf("~a: %d\n", (~a));        // Bitwise NOT (result = -6)
    printf("a << 1: %d\n", (a << 1)); // Left shift (result = 10)
    printf("a >> 1: %d\n", (a >> 1)); // Right shift (result = 2)
    return 0;
}

5. Assignment Operators

Assignment operators are used to assign values to variables.

  • =: Simple assignment
  • +=: Add and assign
  • -=: Subtract and assign
  • *=: Multiply and assign
  • /=: Divide and assign
  • %=: Modulus and assign

Example:

#include <stdio.h>
int main() {
    int a = 10;
    a += 5; // a = a + 5
    printf("a += 5: %d\n", a);
    
    a -= 3; // a = a - 3
    printf("a -= 3: %d\n", a);
    
    a *= 2; // a = a * 2
    printf("a *= 2: %d\n", a);
    
    a /= 4; // a = a / 4
    printf("a /= 4: %d\n", a);
    
    a %= 3; // a = a % 3
    printf("a %%= 3: %d\n", a);

    return 0;
}

6. Increment and Decrement Operators

Increment and decrement operators are used to increase or decrease a variable's value by 1.

  • ++: Increment operator
  • --: Decrement operator

Example:

#include <stdio.h>
int main() {
    int a = 5;
    printf("a++: %d\n", a++);  // Post-increment
    printf("++a: %d\n", ++a);  // Pre-increment
    printf("a--: %d\n", a--);  // Post-decrement
    printf("--a: %d\n", --a);  // Pre-decrement
    return 0;
}

7. Conditional (Ternary) Operator

The conditional operator is a shorthand for an if-else statement.

  • ?:: Conditional (Ternary) operator

Example:

#include <stdio.h>
int main() {
    int a = 5, b = 3;
    
    // If a is greater than b, result is a, else result is b
    int result = (a > b) ? a : b;
    printf("The greater value is: %d\n", result);
    return 0;
}

Output:

8. Sizeof Operator

The sizeof operator is used to determine the size (in bytes) of a data type or variable.

  • sizeof(type)
  • sizeof expression

Example:

#include <stdio.h>
int main() {
    int a = 10;
    double b = 3.14;
    char c = 'A';
    printf("Size of int: %zu bytes\n", sizeof(a));
    printf("Size of double: %zu bytes\n", sizeof(b));
    printf("Size of char: %zu byte\n", sizeof(c));
    return 0;
}

9. Comma Operator

The comma operator allows you to perform multiple expressions in a single statement. It evaluates each expression from left to right and returns the value of the last expression.

Example:

#include <stdio.h>
int main() {
    int a = 1, b = 2, c = 3;
    // Using the comma operator
    int result = (a++, b++, c++);
    printf("a: %d, b: %d, c: %d, result: %d\n", a, b, c, result);
    return 0;
}
