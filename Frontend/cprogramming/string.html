Strings in C Programming

In C programming, a string is a sequence of characters terminated by a null character ('\0'). Strings are handled as arrays of characters. Each element of the array is a character, and the null character marks the end of the string.

1. Declaring and Initializing Strings

A string can be declared as a character array or can be initialized using string literals.

Syntax:

char string_name[size];
OR
char string_name[] = "string_literal";
  • string_name[]: An array of characters.
  • size: The size of the array (can be omitted when initializing with a literal).
  • "string_literal": The string literal automatically includes the null character ('\0').

Example: String Declaration and Initialization

#include <stdio.h>
int main() {
    // Declaring a string using a character array
    char str1[] = "Hello, World!";
    
    // Declaring a string with a fixed size
    char str2[20] = "Hello, C!";
    
    // Printing the strings
    printf("str1: %s\n", str1);
    printf("str2: %s\n", str2);
    
    return 0;
}

2. String Input and Output

Strings in C can be inputted using the scanf() function or gets() (though gets() is discouraged due to potential buffer overflow risks). For safer input, fgets() is recommended.

Example: Using scanf and fgets to take input:

#include <stdio.h>
int main() {
    char str[100];
    
    // Using scanf (unsafe for strings with spaces)
    printf("Enter a string using scanf: ");
    scanf("%s", str);  // Only takes a single word, not spaces.
    printf("You entered: %s\n", str);
    
    // Using fgets (safe for strings with spaces)
    printf("Enter a string using fgets: ");
    fgets(str, 100, stdin);  // Reads an entire line, including spaces.
    printf("You entered: %s\n", str);
    
    return 0;
}

  • fgets reads an entire line (including spaces) into str until a newline or the buffer size is reached.

3. String Functions in C

C provides several built-in functions to manipulate strings. These functions are available in the <string.h> library.

Common String Functions:

  • strlen(str) - Returns the length of the string (not including the null character).
  • strcpy(destination, source) - Copies the string from source to destination.
  • strcat(destination, source) - Concatenates source string to the end of destination.
  • strcmp(str1, str2) - Compares two strings lexicographically.
  • strchr(str, character) - Finds the first occurrence of character in str.

Example: Using String Functions

#include <stdio.h>
#include <string.h>
int main() {
    char str1[50] = "Hello";
    char str2[] = "World!";
    char str3[50];
    
    // Copying str2 to str3
    strcpy(str3, str2);
    printf("str3 after strcpy: %s\n", str3);
    
    // Concatenating str2 to str1
    strcat(str1, str2);
    printf("str1 after strcat: %s\n", str1);
    
    // Length of str1
    printf("Length of str1: %lu\n", strlen(str1));
    
    // Comparing str1 and str2
    int result = strcmp(str1, str2);
    if (result == 0) {
        printf("str1 and str2 are equal.\n");
    } else if (result < 0) {
        printf("str1 is less than str2.\n");
    } else {
        printf("str1 is greater than str2.\n");
    }
    
    // Finding a character in str1
    char *ptr = strchr(str1, 'o');
    if (ptr != NULL) {
        printf("Character 'o' found at position: %ld\n", ptr - str1);
    }
    
    return 0;
}

4. String Searching

String searching involves looking for a substring or a specific character in a string. The two most common methods for searching in C are strstr (for substring) and strchr (for a specific character).

Example: Using strstr and strchr for Searching

#include <stdio.h>
#include <string.h>
int main() {
    char str[] = "Programming in C is fun!";
    
    // Search for a substring
    char *substr = strstr(str, "C");
    if (substr != NULL) {
        printf("Found substring 'C' at position: %ld\n", substr - str);
    } else {
        printf("Substring not found.\n");
    }
    
    // Search for a character
    char *charPos = strchr(str, 'i');
    if (charPos != NULL) {
        printf("Found character 'i' at position: %ld\n", charPos - str);
    } else {
        printf("Character not found.\n");
    }
    
    return 0;
}

  • strchr searches for the character 'i' and returns a pointer to its first occurrence.

Found character 'i' at position: 2

5. String Comparison

String comparison is used to compare two strings lexicographically. The function strcmp is used for this purpose.

Example: Comparing Two Strings

#include <stdio.h>
#include <string.h>
int main() {
    char str1[] = "Apple";
    char str2[] = "Orange";
    
    int result = strcmp(str1, str2);
    if (result == 0) {
        printf("The strings are equal.\n");
    } else if (result < 0) {
        printf("str1 is lexicographically smaller than str2.\n");
    } else {
        printf("str1 is lexicographically larger than str2.\n");
    }
    
    return 0;
}

Dry Run:

Output:
