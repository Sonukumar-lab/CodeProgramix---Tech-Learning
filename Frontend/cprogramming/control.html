Control Structures in C

Control structures in C are used to control the flow of execution of a program. They enable the program to make decisions, repeat certain operations, and branch to different parts based on conditions. There are three primary types of control structures in C:

  • 1. Decision-Making Statements (Conditional statements)
  • 2. Looping Statements (Repetition statements)
  • 3. Jump Statements

Let's explore each category with examples, followed by questions and dry runs.

1. Decision-Making Statements

Decision-making statements are used to make decisions based on certain conditions.

a. if Statement

The if statement is used to test a condition. If the condition is true, the code inside the if block is executed.

#include <stdio.h>
int main() {
    int a = 10;
    if (a > 5) {
        printf("a is greater than 5\n");
    }
    return 0;
}

b. if-else Statement

The if-else statement provides two paths: one if the condition is true and the other if the condition is false.

#include <stdio.h>
int main() {
    int a = 4;
    if (a > 5) {
        printf("a is greater than 5\n");
    } else {
        printf("a is less than or equal to 5\n");
    }
    return 0;
}

Dry Run:

  • a = 4, the condition a > 5 is false, so "a is less than or equal to 5" is printed.

Output:

a is less than or equal to 5

c. if-else-if Ladder

The if-else-if ladder is used to test multiple conditions. The first true condition will execute, and the rest are skipped.

 #include <stdio.h>
int main() {
    int a = 15;
    if (a > 20) {
        printf("a is greater than 20\n");
    } else if (a == 15) {
        printf("a is equal to 15\n");
    } else {
        printf("a is less than 15\n");
    }
    return 0;
}

Dry Run:

  • a = 15, the first condition a > 20 is false, the second condition a == 15 is true, so "a is equal to 15" is printed.

Output:

a is equal to 15

2. Looping Statements

Looping statements are used when we want to execute a block of code multiple times.

a. while Loop

The while loop repeatedly executes a block of code as long as the condition is true.

#include <stdio.h>
int main() {
    int i = 1;
    while (i <= 5) {
        printf("%d\n", i);
        i++;
    }
    return 0;
}

b. for Loop

The for loop is used to execute a block of code a specific number of times.

#include <stdio.h>
int main() {
    for (int i = 1; i <= 5; i++) {
        printf("%d\n", i);
    }
    return 0;
}

c. do-while Loop

The do-while loop guarantees at least one execution of the loop body because the condition is checked after executing the loop body.

#include <stdio.h>
int main() {
    int i = 1;
    do {
        printf("%d\n", i);
        i++;
    } while (i <= 5);
    return 0;
}

3. Jump Statements

Jump statements are used to control the flow of execution in a program.

a. break Statement

The break statement is used to exit from a loop or a switch statement.

#include <stdio.h>
int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            break;  // Exits the loop when i is 3
        }
        printf("%d\n", i);
    }
    return 0;
}

Dry Run:

Output:

b. continue Statement

The continue statement skips the current iteration of a loop and moves to the next iteration.

#include <stdio.h>
int main() {
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            continue;  // Skips the iteration when i is 3
        }
        printf("%d\n", i);
    }
    return 0;
}

c. Switch Statement

The switch statement allows multi-way branching based on the value of a variable.

#include <stdio.h>
int main() {
    int a = 2;
    switch (a) {
        case 1:
            printf("Case 1\n");
            break;
        case 2:
            printf("Case 2\n");
            break;
        case 3:
            printf("Case 3\n");
            break;
        default:
            printf("Default case\n");
    }
    return 0;
}

Dry Run:
