Functions in C & Recursion: Call by Value and Call by Reference

In C programming, functions are used to organize code into reusable blocks that can be executed when called. Functions help to reduce redundancy, improve code readability, and manage larger programs effectively. There are two main concepts related to functions:

  • 1. Call by Value
  • 2. Call by Reference

Additionally, recursion is a key concept in programming where a function calls itself to solve a problem. Let's explore each concept with examples, followed by some practice questions.

1. Functions in C

A function in C is a block of code that performs a specific task. It allows you to divide your program into smaller, modular, and reusable pieces.

Syntax:

return_type function_name(parameter_list) {
    // body of the function
}
  • return_type: Type of value the function returns (e.g., int, float, void).
  • function_name: Name of the function.
  • parameter_list: List of parameters passed to the function (optional).

Example:

#include <stdio.h>
void greet() {
    printf("Hello, World!\n");
}
int add(int a, int b) {
    return a + b;
}
int main() {
    greet();  // Calling the greet function
    int sum = add(5, 3);  // Calling the add function
    printf("Sum: %d\n", sum);
    return 0;
}

2. Call by Value

In call by value, the actual value of the arguments is passed to the function. This means that the function works on the copies of the actual arguments, and changes made to the parameters inside the function do not affect the original arguments.

Syntax:

return_type function_name(parameter_list) {
    // body of the function
    // modifies only the local copies of arguments
}

Example:

#include <stdio.h>
void modifyValue(int a) {
    a = a + 10;  // This will not affect the original variable
    printf("Inside function: %d\n", a);
}
int main() {
    int num = 5;
    printf("Before function call: %d\n", num);
    modifyValue(num);  // Passing value of num
    printf("After function call: %d\n", num);  // Value of num remains unchanged
    return 0;
}

Dry Run:

  • num is passed as 5 to modifyValue().
  • Inside modifyValue(), the local variable a becomes 15, but this change does not affect num in the main function.
  • Therefore, num remains 5 after the function call.

Output:

Before function call: 5
Inside function: 15
After function call: 5

3. Call by Reference

In call by reference, the memory address (reference) of the actual argument is passed to the function. This means that the function works on the actual variables, and changes made inside the function will affect the original arguments.

Syntax:

return_type function_name(parameter_list) {
    // body of the function
    // modifies the actual arguments passed to the function
}

Example:

#include <stdio.h>
void modifyValue(int *a) {
    *a = *a + 10;  // This will modify the actual variable
    printf("Inside function: %d\n", *a);
}
int main() {
    int num = 5;
    printf("Before function call: %d\n", num);
    modifyValue(&num);  // Passing address of num
    printf("After function call: %d\n", num);  // Value of num is modified
    return 0;
}

Dry Run:

  • The address of num is passed to modifyValue().
  • Inside modifyValue(), the value of num is updated to 15 because the function operates on the actual memory location of num.

Output:

Before function call: 5
Inside function: 15
After function call: 15

4. Recursion

Recursion is a technique where a function calls itself to solve smaller instances of the same problem. It is often used to solve problems that can be broken down into smaller, similar sub-problems.

Example (Factorial using recursion):

#include <stdio.h>
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;  // Base case
    } else {
        return n * factorial(n - 1);  // Recursive call
    }
}
int main() {
    int num = 5;
    int result = factorial(num);
    printf("Factorial of %d is %d\n", num, result);
    return 0;
}

Copy and,Test this codeðŸ‘†

Dry Run:

  • factorial(5) is called.
  • This calls factorial(4), which calls factorial(3), and so on, until factorial(0) is reached, which returns 1.
  • Then, the results are multiplied back up the call stack: 5 * 4 * 3 * 2 * 1 = 120.

Output:

Factorial of 5 is 120

Call by Value vs Call by Reference

1. Call by Value:

  • A copy of the actual argument is passed to the function.
  • Modifications inside the function do not affect the actual argument.

2. Call by Reference:

  • The memory address (reference) of the argument is passed to the function.
  • Modifications inside the function affect the actual argument.

Programming Questions:

1. Write a program to swap two numbers using call by value.

#include <stdio.h>
void swap(int a, int b) {
    int temp;
    temp = a;
    a = b;
    b = temp;
    printf("Inside swap function: a = %d, b = %d\n", a, b);
}

int main() {
    int num1, num2;
    printf("Enter the value of num1: ");
    scanf("%d", &num1);
    printf("Enter the value of num2: ");
    scanf("%d", &num2);

    printf("Before swap: num1 = %d, num2 = %d\n", num1, num2);
    swap(num1, num2);

    printf("After swap function call: num1 = %d, num2 = %d\n", num1, num2);

    return 0;
}

2. Write a program to swap two numbers using call by reference.

#include <stdio.h>
void swap(int *a, int *b) {
    int temp;
    temp = *a;
    *a = *b;
    *b = temp;
    printf("Inside swap function: a = %d, b = %d\n", *a, *b);
}

int main() {
    int num1, num2;
    printf("Enter the value of num1: ");
    scanf("%d", &num1);
    printf("Enter the value of num2: ");
    scanf("%d", &num2);

    printf("Before swap: num1 = %d, num2 = %d\n", num1, num2);
    swap(&num1, &num2);

    printf("After swap function call: num1 = %d, num2 = %d\n", num1, num2);

    return 0;
}

3. Write a program to calculate the sum of digits of a number using recursion.

#include <stdio.h>
int sum_of_digits(int n) {
    if (n == 0) {
        return 0;
    } else {
        return (n % 10) + sum_of_digits(n / 10);
    }
}

int main() {
    int num;
    printf("Enter a number: ");
    scanf("%d", &num);
    
    printf("The sum of digits is: %d\n", sum_of_digits(num));
    
    return 0;
}
